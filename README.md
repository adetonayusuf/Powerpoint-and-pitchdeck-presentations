- Home ribbon - where basic formatting functions are contains
- Insert ribbon - you can insert new slides, add table, pictures, shapes, textbox, video, audi0 and icons
- Design ribbon - select slide size - whether standard 4x3 or wwidescreen. Standard is used when printing the presentation as this accommodated most papaer standard and widescreen if it for live presentation. format background
- Transation section - Transition are the way power point move between slides, this is only useful in live presentation or onscreen presentation.
- Animation - Animations are movement within the slides - only useful for live or onscreen presentation.
- Slides show - This is where you actually begin presentation as a presenter giving you a live oresentation
- Review - You cn do your spell check, 
- View - You can easily move between normal views, slide sorter etc. You can edit the slide master  which enables you to set the design for the presentation for example add the company's logo
